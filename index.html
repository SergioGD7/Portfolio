PROJECTS

Here you will find some of the personal and clients projects that I created with each project containing its own case study

Memorion

Memorion is a project made in Java at the beginning of my academic training, in which you have to find the pairs of images in the shortest possible time and with the fewest number of attempts, apart from being able to save and load the game and record the data so that it can be shown in the ranking once the game is finished

VIEW PROJECT

Snake Game

The Snake game is a project made with TypeScript, NextJS, Tailwind CSS in which the player must eat as much food as possible without crashing, in which you can choose between four difficulty levels, you can start/pause the game, you can generate another random scenario, it has a scoring system and a ranking with the ten best scores

VIEW PROJECT

Workout Planner

Workout Planner is a project made with TypeScript, NextJS, Tailwind CSS in which the user can create training, add exercises to the training, modify the series, repetitions and weight of each one, add custom exercises, edit the exercises or delete them, see a graph with the progress according to the selected period of time and access a calendar from which to see the exercises performed each day
